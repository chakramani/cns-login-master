Tested up to: 6.3





== Installation ==

How to install the plugin?

e.g.

1. Upload `cns login master` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'CNS Login Master' and customize your login form.
