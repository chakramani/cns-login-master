Allow users to customize the admin login form. You can also remove unwanted things from form.

== Description ==

This plugin allows you to customize the admin login from. It could be designing the form, removing unwanted things from the login form.

== Installation ==

How to install the plugin?

e.g.

1. Upload `cns login master` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'CNS Login Master' and cutomize your login form.
